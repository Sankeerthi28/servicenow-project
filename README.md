# servicenow-project
ServiceNow Role-Based Access Control Project

Description:
This project demonstrates user, role, group management, ACLs, and workflow automation in ServiceNow.

Steps:
- Created users Alice and Bob
- Created Project Team group
- Created Project Manager and Team Member roles
- Assigned roles to users
- Created task/project table
- Configured ACLs
- Created Flow Designer workflow
- Tested system using both users
